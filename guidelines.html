Black

Color Name & Hex Value

Primary color, --black, #000000

Context of Use

Sitewide

  • Site-nav logo text and icon outline
  • All footer text (span & p)
  • Site-nav notifications icon
  • All search icons

Home Page

  • main headings (h2)
  • main body text (p)

Support & FAQ Page

  • form labels and user input field text (label & input)
  • form text input field borders (input)
  • All accordion text (headings and body text, summary & p)
  • Accordion arrow icons

Search Page

  • Filters dropdown text (select & option)
  • Filters borders (select)
  • Post headings and body text (h3 & p)

Messages Page

  • Vertical nav links and page title (p & h1)
  • Convo-cards titles (p)
  • Select all label (p)

Single-Post Page

  • Page title (h1)
  • create-post user input field text and borders (textarea)
  • Create-post filter dropdown text & borders (input)
  • Add-more-tags tag text (p)
  • Save-as-draft button text (a)

Login Page

  • Page title and subtitle (h1 & p)
  • form user input field labels, text, and borders (label & input)

Rationale

In our design, black fulfills the function it commonly does in many sites. It’s the default color, particularly for large bodies of text. It’s also used to contrast between lighter colors to make everything easier to read and understand.

White

Color Name & Hex Value

Primary color, --white, #FFFFFF

Context of Use

Sitewide

  • Default page background (body)
  • Search bars background (body)
  • header site-navbar background
  • site-navbar create-post button text (input)
  • footer social-media background and icon design
  • All user input fields and dropdowns backgrounds (select, option, & input)
  • Any image-carousel arrow icons
  • Hero page title & subtitle (h1 & p)
  • All filters dropdown backgrounds (select & option)
  • Text for tags & buttons that have coral or teal backgrounds

Home Page

  • 3-col card backgrounds and icon designs (div)
  • community section heading & body text (h2 & p)

Support & FAQ Page

  • All accordion heading backgrounds (details & summary)
  • form card background
  • Support-request section heading & body text (h1 & p)

Search Page

  • Clear-filters and sort-by button text (p)
  • Post-cards tags text (p)
  • Pagination inactive-backgrounds & active-text

Messages Page

  • Convo-cards backgrounds (div)

Rationale

White tends to function well as a background against darker images and text, and just as easily stands out as a small icon or text when put against a darker background.

Teal

Color Name & Hex Value

Primary color, --teal-main, #016F75

Context of Use

Sitewide

  • Site-nav logo design
  • footer social-media icon designs
  • Some tags and button backgrounds (input & p)

Home Page

  • 3-col card borders and icon designs (div)
  • community section background (div)

Support & FAQ Page

  • Accordion subheading borders and heading underline (p & details)
  • Support-request section background (div)
  • form submit button background (input)

Search Page

  • Pagination active-background, inactive-icons, and inactive-text

Rationale

Teal functions as a darker color to separate and accent content, and to just add some more color to the site overall. It’s also used to draw the eye towards certain parts of the page, and occasionally signify that a component is interactive.

Icon Library

Name & Description

Font Awesome

  • Search icon
  • Search icon thin
  • Question mark
  • Dollar sign
  • Chevron right
  • Chevron left

Rationale

Font Awesome icons are easily accessible; they are easily added to any project with an i tag and a link to a stylesheet in the head of an HTML document. Moreover, these icons are easy to edit and come in many different thicknesses. They feature clear designs that are indicative of their name- this allows for users to easily understand what the developers and designers are intending to communicate.

Recommendations & How-Tos

Use an i tag to place Font Awesome icons in your project.
